# 🎉 Event Management Website

A dynamic web application built using Flask that allows users to browse events, register online, and enables administrators to manage events efficiently.

---

## 📖 Project Overview

The Event Management Website is designed to provide a simple and user-friendly platform where users can explore various events and register for them. It also includes an admin panel to add and delete events.

This project demonstrates the integration of front-end and back-end technologies using Flask and Jinja2 templates.

---

## 🚀 Tech Stack

### Front-End
- HTML5
- CSS3
- JavaScript

### Back-End
- Python (Flask)

### Template Engine
- Jinja2

---

## ✨ Features

- 📅 View all available events
- 📝 Register for events
- 🔐 Admin panel to manage events
- ➕ Add new events
- ❌ Delete events
- 💻 Responsive and clean UI
